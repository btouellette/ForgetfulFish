# Agent Instructions

All AI coding agents working in this repository must follow:

- `docs/AI_TOOLING_RULES.md`

## Default Expectations
- Use test-first development for new functionality.
- Add a failing regression test before implementing bug fixes.
- Keep changes scoped and deterministic.
- Update docs and decisions when behavior or architecture changes.
- Run relevant tests before declaring completion (or explicitly report what was not run).
